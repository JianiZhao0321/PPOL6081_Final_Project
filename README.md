# PPOL6081_Final_Project

## Overview

This repository contains the materials for my PPOL 6081 final project, which examines gender differences in central bank communication using dictionary-based tone measures and semantic analysis. The project uses central bank speech data to study how male and female policymakers differ in their use of language when discussing monetary policy.

## Input
export_lexicon.csv: Contains a dictionary of hawkish and dovish terms used to measure monetary policy tone in speeches

## Code
Main_RQ.qmd: Documents the research question and analytical approach

## Output
output/: Results generated by Main_RQ.qmd

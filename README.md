# PPOL6081_Final_Project

## Overview

This repository contains the materials for my PPOL 6081 final project, which examines gender differences in central bank communication. The project uses dictionary-based tone measures and semantic analysis to study how male and female policymakers differ in their use of language when discussing monetary policy.

## Input

- **Central Bank Speeches dataset**  
  Source: https://cbspeeches.com/  
  Provides speech text and metadata used for the analysis.

- **export_lexicon.csv**  
  Contains a dictionary of hawkish and dovish terms used to measure monetary policy tone in speeches.

## Code

- **Main_RQ.qmd**  
  Documents the research question and analytical approach.  
  Performs text preprocessing, dictionary-based tone analysis, and semantic analysis.

## Output

- **output/**  
  Contains figures and results generated by `Main_RQ.qmd`.

## Report and Proposal

- **PPOL6081_Final_Project_Report.pdf**  
  Final project report presenting motivation, data, methods, results, and discussion.

- **Final Project Proposal.pdf**  
  Original project proposal outlining the research question and planned methods.
